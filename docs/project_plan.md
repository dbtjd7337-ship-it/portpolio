# Portfolio Project Planning

이 폴더는 프로젝트의 기획, 디자인 시스템, 로직 설계 및 기술 문서를 정리하는 공간입니다.

## 📂 폴더 구조

- `docs/`
  - `typography.md`: 사이트의 폰트 스타일 및 타이포그래피 컴포넌트 정의
  - `project_plan.md`: 전체 프로젝트 기획 및 로드맵 (현재 문서)

## 🎯 프로젝트 목표

1. **시각적 압도감**: 3D 시각화와 인터랙티브 요소를 결합한 프리미엄 포트폴리오 구축.
2. **논리적 설계**: 디자인 배경과 개발 로직이 결합된 'Logic Architect'로서의 역량 강조.
3. **사용자 경험**: 스크롤 애니메이션과 반응형 레이아웃을 통한 매끄러운 내비게이션 제공.

## 🛠 기술 스택

- **Core**: React, TypeScript, Vite
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Interaction**: Three.js (TBD)
